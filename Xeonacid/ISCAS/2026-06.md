### 本月工作总结

Xeonacid 这个月的贡献中 worktrunk 的两个合并 PR 质量最高，一个拆清了 Git 临时 index 在慢速 riscv64 机器上漏报同尺寸文件修改的原因，并调整了 untracked diff 统计路径；另一个处理了 `posix_spawn` 在 qemu-user、OpenBSD、HPPA 等环境下可能丢失 child-side exec 错误的问题，补了 preflight 检查和回归测试。moxcms 的固定点 LUT 实现补齐了非 SIMD fallback 和 SIMD 路径的行为差异，解释并修复了 Typst 在 riscv64 上暴露的一位色差。TooManyCooks、OpenRCT2 和 snmalloc 的工作集中在架构支持、构建解锁和运行时诊断，其中 snmalloc issue 对 Sv39/Sv48 地址空间问题给出了清楚的复现数据和方案边界。
